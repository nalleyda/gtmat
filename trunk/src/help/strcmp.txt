Inputs:
    1. string (char)
    2. string (char)

Outputs:
    1. true or false (logical)

Form:
    TorF=strcmp(str1, str2)

Description:
    Compares two strings; returns true if equal.

Examples:
    TorF=strcmp('Jolly', 'Molly');
    %TorF=f

    TorF=strcmp('La Di Da', 'La Di Da');
    %TorF=t
